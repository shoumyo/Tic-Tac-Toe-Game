  <h3>Rules of Engagement</h3>
  <p>
    The game is played on a grid of 9 squares. Two players alternate turns:
  </p>
  <ul>
    <li>Player 1 uses <b>O</b></li>
    <li>Player 2 uses <b>X</b></li>
  </ul>
  <p>
    The player using <b>O</b> selects an empty square first, followed by the player using <b>X</b>. 
    Play continues until a winner is declared or the grid is full.
  </p>

  <h3>Win Conditions</h3>
  <ul>
    <li>
      <strong>Victory:</strong> The first player to align three of their marks horizontally, 
      vertically, or diagonally wins the match.
    </li>
    <li>
      <strong>Draw:</strong> If all squares are filled and no player has achieved a winning 
      alignment, the game results in a draw.
    </li>
  </ul>
